## 1. Présentation générale

**Fonetix** est une plateforme en ligne d'autoapprentissage de la prononciation du français, conçue par des enseignants de FLE (Français Langue Étrangère) expérimentés. Projet soutenu par la **Délégation générale à la langue française et aux langues de France**.

### Proposition de valeur

- Correction de la prononciation = "parent pauvre" de l'enseignement du FLE
- La prononciation est évaluée lors des tests officiels de français
- La maîtrise des sons est un préalable à l'accès à l'écrit
- **20 minutes par jour** suffisent pour progresser
- Intégration d'**IA** pour un retour objectif sur la **compréhensibilité** du français oral

### Public cible

- **Étudiants** : apprenants de français de tous niveaux, toutes langues maternelles
- **Enseignants** : écoles et organismes de formation FLE (abonnements, parcours sur mesure)

## 2. Approche pédagogique

### Fondement : la Méthode Verbo-Tonale (MVT)

Méthode issue de l'université Toulouse 2 (enseignements de **Michel Billières**). Grands principes :

1. **Priorité à la prosodie** (rythme, intonation, accentuation)
2. **Progression réfléchie** dans les activités
3. **Centration sur les difficultés** spécifiques de chaque apprenant

### Principe clé : perception avant production

> « Une prononciation erronée en langue étrangère s'explique par une mauvaise perception et doit se travailler non pas avec la bouche, mais avec l'oreille ! C'est en percevant mieux qu'on prononce mieux. »

Notre cerveau est conditionné dès le plus jeune âge à analyser les sons de notre langue maternelle. Cette spécialisation nous rend "sourds" aux sons qui n'appartiennent pas à notre propre système phonétique — on les assimile à ceux de notre langue. Au moment de reproduire un énoncé, on le fait sur la base de ce qu'on a "mal" perçu : c'est l'accent étranger.

### Parcours différenciés par langue maternelle

Chaque langue maternelle présente des difficultés de prononciation spécifiques. Fonetix propose des parcours adaptés :

- **Disponibles** : espagnol, anglais, arabe, chinois, vietnamien
- **À venir** : russe, portugais
- Une première partie commune (gratuite) est accessible à tous

### Progression logique

Parcours progressif : chaque étape consolide les acquis et prépare la suivante. Difficulté croissante mais accessible.

**Ordre de travail** :
1. **Prosodie d'abord** : rythme (organisation des syllabes), accentuation (syllabes importantes), intonation (modulations de la voix)
2. **Sons ensuite** : consonnes, voyelles, semi-consonnes — s'intègrent dans le "moule" prosodique

## 3. Système phonétique du français

### Voyelles (15 phonèmes)

| Antérieures non-arrondies | Antérieures arrondies | Postérieures | Nasales |
|---|---|---|---|
| i | y | u | |
| e | ø | o | |
| ɛ | œ | ɔ | |
| a | | | |
| | | ə (schwa) | |
| | | | ɛ̃, œ̃, ɔ̃, ɑ̃ |

### Consonnes

| Type | Sourdes | Sonores | Nasales/Liquides |
|---|---|---|---|
| **Explosives (occlusives)** | p, t, k | b, d, g | m, n |
| **Fricatives (constrictives)** | f, s, ʃ | v, z, ʒ | l, ʁ |
| **Glissantes (semi-consonnes)** | | w, ɥ, j | |

### Syllabes

- Une syllabe = minimum une voyelle (phonétique, pas orthographique)
- Exemple : "oiseau" [wazo] = 5 voyelles orthographiques mais 2 voyelles phonétiques → 2 syllabes
- Les syllabes du français sont généralement **ouvertes** (se terminent par une voyelle)

### Enchaînement consonantique

Quand un mot se termine par une consonne prononcée, celle-ci forme une syllabe avec la voyelle initiale du mot suivant.

- "Il dort mal" → [il-dɔʁ-mal]
- "Il dort ici" → [il-dɔ-**ʁi**-si]

### Enchaînement vocalique

Passage d'une voyelle à une autre produit de manière continue, sans pause silencieuse.

- "Elle va à Albi ?" → [ɛl-va-a-al-bi]

## 4. Rythme, intonation & accentuation

### Rythme — Isosyllabisme

Le français est caractérisé par l'**isosyllabisme** : toutes les syllabes ont à peu près la même durée, **sauf la dernière syllabe d'un groupe rythmique** qui dure 2 fois plus longtemps.

Un **groupe rythmique** = ensemble de syllabes constituant un groupe de signification.

Exemple : {Mes amis japonais} {visitent Albi}

### Intonation

Variation de la hauteur de la voix (grave/aigu) en fin de groupe :
- **Intonation démarcative** : délimite les groupes
- **Intonation sémantique** : donne un sens particulier

### Accentuation

En français, la **dernière syllabe** d'un groupe est accentuée :
- Dure 2 fois plus longtemps
- Porte l'intonation (montée ou descente)

## 5. Structure des exercices

### Format pédagogique

1. **Écoute** d'énoncés modèles
2. **Répétition** avec enregistrement
3. **Comparaison** avec le modèle
4. **Quiz de perception** pour valider les acquis

La difficulté augmente progressivement. Environnement ludique (sonore et visuel).

### Rôle de l'IA

- Retour objectif sur la **compréhensibilité** des productions orales
- Complément du savoir-faire des formateurs (ne remplace pas l'expertise humaine)
- Feedback instantané sur la clarté et l'intelligibilité

## 9. Points clés pour le projet Diction Coach

### Ce qu'on peut reprendre/adapter

1. **Approche perception-first** : travailler l'oreille avant la bouche — principe fondamental transposable
2. **Parcours par langue maternelle** : chaque L1 a des difficultés spécifiques en français
3. **Prosodie d'abord, sons ensuite** : progression validée par 10+ ans de pratique
4. **Boucle écoute → répétition → comparaison → quiz** : format d'exercice éprouvé
5. **IA pour feedback compréhensibilité** : pas de remplacement humain, enrichissement
6. **Isosyllabisme + accentuation finale** : caractéristiques du rythme français à intégrer dans tout coaching diction
7. **Enchaînements** (consonantique + vocalique) : points critiques de la fluidité en français
8. **Tarification simple** : 16-72 EUR, accès illimité sur la durée

### Différenciation possible

- Fonetix cible les **apprenants FLE** (français langue étrangère) — un diction coach pourrait cibler les **francophones natifs** voulant améliorer leur élocution professionnelle
- Fonetix est une plateforme web avec exercices pré-conçus — un coach vocal IA pourrait offrir des sessions conversationnelles en temps réel
- L'approche MVT est très académique — un produit grand public pourrait vulgariser ces principes
